SpiralTorch Mellin Log Grid

WASM demo for MellinLogGrid.evaluateMany (log-uniform samples → Mellin transform).

Log-uniform samples

Samples are generated in WASM using f(x)=exp(-x) on x_k = exp(log_start + k·log_step).

Evaluation points

Use mode=vertical to plot a single vertical line. Use mode=mesh to render a heatmap over real_min..real_max × imag_min..imag_max.

Learning (match reference)

Creates a second (“learnable”) sample grid and takes gradient steps so its Mellin transform matches the reference exp(-x) grid along the current vertical evaluation line.
